# PowerLevel10K v1.4-BETA
PowerLevel10K with zsh shell for Termux. Most customizable Termux shell theme. Stylish Shell.

## How to use(commands):
```
cd $HOME
git clone https://github.com/AbirHasan2005/PowerLevel10K
cd PowerLevel10K
chmod +x setup.sh
bash setup.sh
```
